# JsonMocker
An easy to use web-based editor to create reusable json mock data schema files.

![Screenshot](docs/img.png)

[Try it out now!](https://laurawebdev.github.io/json-mocker)

## Development
- `npm install`
- `npm run dev`

## Building
- `npm install`
- `npm run build`
